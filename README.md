# Jogo da velha (tic-tac toe)

Esse jogo faz parte da lista de exercícios do curso de Python do grande professor
Gustavo Guanabara.
Como sou iniciante em programação, não consegui fazer com que duas pessoas jogassem via internet.
Por isso, cada jogador precisa usar alternadamente o mesmo teclado. Não é possível usar o mouse.
O jogo roda exclusivamente pelo terminal, sendo necessário baixá-lo e executá-lo para se jogar.

> [!IMPORTANT]
> O programa apresenta um bug que eu não consegui corrigir. Ele não detecta vitórias.
