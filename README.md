# Jogo da Velha

Aqui eu apresento o meu jogo da velha (tic-tac-toe) refeito do zero pelo celular.
Agora ele detecta a vitória corretamente.

## Como jogar?
1. O jogo é para dois jogadores humanos (ainda não sei trabalhar com inteligência artifical)
2. Cada um joga alternadamente, começando pelo jogador_1, que é o "X". O jogador_2 é o "O".
3. As regras para vencer neste jogo valem as mesmas do jogo tradicional, quem fizer uma linha reta com três casas primeiro, ganha

[!IMPORTANT]
Esse jogo só roda no terminal do Python.
